# 🎓 TUT Event Management System  

A comprehensive web-based platform developed by **TriTech Solutions** for the **Tshwane University of Technology (TUT)**.  
The system enables organizers to easily create and manage university events, while students can explore, register, and attend events using secure **QR-code eTickets**.

🔗 **Live Demo:** [https://mytutevents.vercel.app/](https://mytutevents.vercel.app/)

---

## 🚀 Features  

### 🔹 Admin / Organizer Features  
- 🔐 **Secure Authentication:** Login with staff ID and password using Flask sessions and bcrypt hashing.  
- 🗓️ **Event Management:** Create, update, and delete events with details (title, date, time, venue, and description).  
- 🖼️ **Media Uploads:** Upload event posters and flyers to promote events visually.  
- 📢 **Notifications:** Send updates to students via:  
  - 📧 Email (SMTP integration)  
  - 💬 WhatsApp messages (WhatsApp API)  
- 📊 **Organizer Dashboard:**  
  - Manage upcoming and past events  
  - View event analytics and attendance statistics  

---

### 🔹 Student Features  
- 🏠 **Dynamic Landing Page:** Displays event highlights and important information.  
- 🔐 **Secure Login & Signup:** Encrypted passwords and session-based authentication.  
- 🧭 **Event Exploration:**  
  - Browse current, upcoming, and past events  
  - Explore event galleries  
  - Discover TUT event organizers  
  - Learn more about the MyTUT platform  
- 🎟️ **Event Registration:** Students register online and receive a **QR-code ticket** for easy event check-in.  

---

### 🔹 Additional Features  
- 🎨 **Modern UI:** Dark-themed interface with brand colors *(dark blue, dark red, dark yellow)*.  
- ⚡ **Interactive Design:** Smooth animations, hover effects, and transitions.  
- 📱 **Responsive Layout:** Works on desktop, tablet, and mobile devices.  
- 🔎 **Search & Filter:** Quickly find events using intuitive filters.  
- 👤 **User Profiles:** Personalized dashboards for students and organizers.  
- 🛡️ **Admin Security:** Only verified TUT organizers can post events.
